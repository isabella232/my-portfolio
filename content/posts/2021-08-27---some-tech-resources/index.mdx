This is just a page that acts as a bookmark to some of my favorite resources. I'll keep updating this list as time goes on.

- Excellent diagramming tool: https://excalidraw.com/
- Create presentations using Markdown in VSS: https://marketplace.visualstudio.com/items?itemName=nicoespeon.slides
- Converting images to the new `webp` format: https://squoosh.app/
- Testing web pages for performance and vulnerabilities: https://www.webpagetest.org
- Top Hackernews articles: https://hn.lindylearn.io/best
- Lots of dev utilities like base64, prettify etc: https://gchq.github.io/CyberChef or https://devutils.app/
- Capture tasks and finish them. Has an offline, local-only version: https://ballertodo.com/local
